## 오늘 공부한 것
- 입출력(scanf/printf 서식지정자), 수식과 연산자

## 핵심 개념
- `%04d` : 자릿수를 4자리로 맞추고 빈 자리는 0으로 채워 출력
- `scanf`에서는 `float`는 `%f`, `double`은 `%lf`로 **반드시 구분**해야 함 
- `printf`에서는 `float`/`double` 둘 다 `%f`로 출력 가능, `double`은 `%lf`도 가능
- 서식지정자 정리: 8진수 `%o`, 10진수 `%u`, 10진수 `%d`, 16진수 `%x`
- 정수형과 부동소수형이 함께 연산되면 정보 손실을 막기 위해 부동소수형으로 자동 변환됨 (피연산자가 모두 정수여도 형변환을 통해 부동소수 연산으로 바꿀 수 있음)
- 복합대입연산자: `a *= b + 3` → `a = a * (b + 3)`
- 논리연산자 우선순위: `!` > `&&` > `||`
- 관계연산자: 좌우 값의 대소관계를 비교하며, 참이면 1, 거짓이면 0 반환
- 반면 `!`의 우선순위는 관계/산술연산자보다 높음 (가장 먼저 적용됨)
- `sizeof` 연산자는 바이트 단위로 크기를 계산
- `(int)2.6`처럼 실수를 정수로 형변환할 때는 반올림이 아니라 소수점 이하를 버림(truncate)하는 규칙 적용 (예: `(int)2.9`도 `2`)

## 막혔던 부분
- 증감연산자: `a++`, `++a` 둘 다 결과적으로 a는 증가하지만, `a++`(후위)는 현재 값을 먼저 사용한 뒤 증가, `++a`(전위)는 먼저 증가시킨 뒤 사용
- 전체 연산자 우선순위 순서: 산술 → 관계 → 논리 → 대입
