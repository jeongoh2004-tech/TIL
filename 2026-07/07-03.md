## 핵심 개념

- 조건문은 조건에 따라 실행되는 코드가 달라지게 할 때 사용하는 제어문
- 조건문 블록에서 중괄호 `{ }` 쌍으로 표현하고, 문장이 하나인 경우는 중괄호 쌍을 생략할 수 있음
- 조건문 안에 조건문을 중첩해서 사용하는 경우를 다중 if문이라고 함
- 다중 if문에서 else는 짝이 없는 위쪽 if 중 가장 가까운 if와 짝이 됨
- switch문은 수식의 결과값과 case의 상수값을 차례로 비교하여, 일치하는 case의 문장들부터 쭉 차례로 수행
- default의 의미는 그 외의 경우 → if 조건문의 else와 같은 느낌
- case에는 정수형 상수(문자 포함)만 가능, 문자열은 안 됨
- break문은 switch문을 중간에 벗어나기 위해 사용함


## 복습 포인트

- 여러 case가 같은 동작을 할 때는 연속해서 작성 (예: case 9: case 10: printf("A\n"))
- grade = (avg >= 75) ? 0 : (avg >= 50 ? 1 : 2) -> 75점 이상이면 0, 아니면 50점 이상인지 다시 물어봐서 1 또는 2를 정하는 구조
