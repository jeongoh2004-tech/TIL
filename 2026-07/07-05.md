## 핵심 개념

**반복문**
- 반복문은 특정 조건을 만족할 때까지 계속 반복하여 실행하는 문장을 의미하며, 간혹 루프(loop)라고 불리기도 함
- `do while`문은 조건식 다음에 세미콜론을 반드시 써야 함
- `while`문은 조건을 먼저 검사하므로 0번 이상 실행된다
- 반복문 안에 또 다른 반복문이 오는 경우를 중첩 반복이라 함
- 다른 종류의 반복문도 중첩 가능 (예: `for`문 안에 `while`문 사용)
- 반복문 안에는 반복되는 것만 넣기
- `break`문은 현재 수행 중인 반복문을 중단하고 지금 반복문을 나감
- `continue`문은 반복문 몸체에서 남은 문장들을 건너뛰고, 바로 조건 검사(또는 증감식) 단계로 넘어가 다음 반복을 수행

**배열**
- 배열이란 동일한 자료형의 변수 여러 개를 하나로 묶은 자료형
- 배열을 구성하는 각 변수를 배열의 원소, `[]` 안의 번호를 첨자라고 함
- 배열의 인덱스는 0부터 시작
- `int i, x[5] = {0, 10, 20, 30, 40};` 형태로 배열 값 초기화
- `int x[5] = {0};` 또는 `int x[5] = {0, };` 로 배열 전체를 0으로 초기화 가능
- 배열 첨자로 연산자를 포함한 수식도 사용 가능

## 복습 포인트

- `do while`문은 실행을 먼저 하므로 1번 이상 실행된다
- 배열의 크기값보다 초기값의 개수가 작으면 배열의 뒷부분은 0으로 초기화됨
- 처음 배열 크기를 지정하지 않으면 초기화에 사용된 원소 수로 크기가 결정됨
- 배열의 원소들이 저장된 장소는 고정되어 있어, 중간에 다른 원소를 추가하거나 빼려면 다른 원소들을 이동시켜야 함
