## 핵심 개념

**함수**
- 함수는 어떤 특정한 일을 수행하는 독립적인 단위
- 함수 정의부(함수를 구현하는 부분, 예: `main()` 함수 구현)와 함수 호출부(함수의 기능을 이용하는 부분, 예: `printf()` 함수 호출)로 나뉨
- 반환형 함수이름 (인자선언1, 인자선언2, …)  
{  
    함수의 수행코드  
    return 문; -> 필요시  
}
- 함수 이름(변수명 규칙과 동일), 매개변수(입력 데이터를 나타내는 변수, 인자가 없어도 소괄호는 반드시 필요), 반환형(수행 후 반환하는 결과의 자료형, 반환값이 없으면 `void`) → 이 세 가지가 함수 헤더
- 함수 몸체는 `{}` 안에 수행할 작업을 명세, `return`문은 함수를 종료하고 결과를 반환(반환형이 `void`면 `return;`만 사용)
- 함수 정의는 보통 호출 위치보다 앞에 작성해야 하지만, 함수 원형(prototype)을 미리 선언해두면 정의가 뒤에 있어도 호출 가능
- 함수 원형 선언 시 인자 이름은 생략 가능, 정의부와 다른 변수명을 써도 됨 (예: `int add(int, int);` 형태 권장)
- 함수에 전달된 값을 실인자, 그 값을 저장하는 변수를 형식인자라 함
- 함수는 코드 작성의 효율성과 가독성을 높여줌

**지역변수 / 전역변수 / 정적변수**
- 지역변수: 함수 내에서 선언, 그 함수 안에서만 유효, 호출과 동시에 생성되고 종료되면 소멸(자동변수라고도 함). 함수의 형식 인자도 지역변수임
- 전역변수: 함수 밖에서 선언, 프로그램 어디서든 사용 가능
- 이름이 겹치면 지역변수가 우선 적용됨(전역변수를 가림)
- 정적변수(`static`): 선언 위치·유효범위는 지역변수와 동일(함수 내부에서만 사용), 지속시간은 전역변수와 동일(프로그램 실행 동안 딱 한 번만 생성·초기화되고, 함수를 여러 번 호출해도 이전 값이 유지됨)

**함수와 배열**
- 함수 원형에서 배열을 전달받을 때 크기를 명시해도 의미 없음(생략 가능)
- 배열을 인자로 넘길 때는 배열 이름만 씀 (예: `sum(a)`)
- `void print_arr(int x[], int n)`처럼 배열과 크기를 같이 전달하고, `sizeof(a)/sizeof(int)`로 배열의 크기를 자동 계산해서 넘길 수 있음
- 호출된 함수에서 배열 값을 변경하면 호출한 쪽의 배열 값도 함께 변경됨

**라이브러리 / 헤더파일**
- 라이브러리는 함수들을 구현해 모아놓은 것, `#include`는 해당 헤더파일을 소스코드에 포함시키라는 의미
- `stdio.h`: 입출력·파일 (`printf`, `scanf`, `putc`, `getc`, `fopen` 등)
- `stdlib.h`: 숫자 변환, 동적 할당, 정수 절댓값 (`atoi`, `rand`, `srand`, `malloc`, `free`, `abs` 등)
- `ctype.h`: 문자 검사 및 변환 (`isalnum`, `isalpha`, `islower`, `toupper` 등)
- `math.h`: 수학 함수, 실수 절댓값 (`sin`, `asin`, `exp`, `log`, `pow`, `sqrt`, `fabs` 등)
- `time.h`: 시간 처리 (`clock`, `time`, `difftime` 등)
- `string.h`: 문자열, 메모리 블록 (`strcpy`, `strcat`, `strcmp`, `strlen`, `memcpy` 등)

**다차원 배열**
- 첨자가 두 개 이상인 배열
- 2차원 배열 선언: `자료형 변수명[크기][크기]`
- 각 원소는 일반 변수 하나와 동일하게 취급됨
- 중첩 중괄호로 행별 초깃값 설정 가능 (예: `int x[3][5] = {0};`)
- 배열 크기를 초깃값 개수로 정할 때, 첫 번째 첨자만 생략 가능하고 두 번째 첨자는 생략 불가 (예: `int x[][2] = {{0,1},{0},{0}};` → `x[3][2]`와 동일)
- 함수의 형식 인자에서는 다차원 배열의 첫 번째 첨자를 보통 생략함
